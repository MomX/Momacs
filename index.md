# Momacs

Interactive Shiny application for digitizing morphometric data from
images.

Supports landmarks, polylines, polygons, curves, and outlines with
export to JSON format.

## Installation

You can install the development version of Momacs from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("MomX/Momacs")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(Momacs)
## basic example code
```
